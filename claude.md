# Tech Stack
- TypeScript
- Vercel
- React.js
- Magic link for userAuth
- Backend is a postgreSQL database

# Project Description
This repo contains the code for the TeraVibes Carpool App, a web application built using TypeScript, Vercel, React.js, and Magic link for user authentication. The app allows users to create and join carpool trips, share ride details, and communicate with fellow riders.
## Features
- User registration and login using Magic link authentication
- Create and join carpool trips
- Share ride details with fellow riders
- Status updates for carpool trips

## User Stories
- As a user, I am either looking to find a carpool trip or create one.
- As a user, I want to be able to register and login to the app using a magic link.
- As a user, I want to be able to create a new carpool trip and invite other users to join.
- As a user, I want to be able to share ride details with other users in the carpool trip.
- As a user, I want to be able to update the status of the carpool trip.

## Look and Feel
- As a user, I want the app to have a clean and modern design.
- As a user, I want the app to be responsive and work well on different devices.
- As a user, I want the app to have a consistent color scheme and typography.

## UX
- As a user, I want the app to have a clear and intuitive navigation - starting with a homepage that asks if I'm looking for a carpool trip or creating one, then prompting me to enter my starting location and date of travel, then moving to authentication.
- As a user, I want the app to have a consistent layout and design across all pages.
- As a user, I want the app to have a responsive design that adapts to different screen sizes.
- As a user, I want the app to have a consistent color scheme and typography.
- As a user, I want the app to have a clear and concise error messaging system.
